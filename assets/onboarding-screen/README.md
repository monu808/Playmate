# Onboarding Images

## Requirements

Please add **3 high-quality images** to this folder with the following names:

1. **slide1.jpg** - For "Find Your Perfect Turf" slide
2. **slide2.jpg** - For "Book Instantly" slide  
3. **slide3.jpg** - For "Play & Enjoy" slide

### Image Specifications:
- **Format**: JPG, PNG, or any format supported by React Native
- **Dimensions**: Recommended 1080x1920 (portrait) or higher
- **Aspect Ratio**: 9:16 (full screen mobile)
- **File Size**: Keep under 2MB each for optimal performance
- **Content**: Make sure images are vibrant and high-quality

### Image Guidelines:
- Slide 1: Show turf browsing/searching (e.g., multiple turfs, sports field, search interface)
- Slide 2: Show booking/payment process (e.g., calendar, payment, reservation)
- Slide 3: Show players enjoying the game (e.g., people playing sports, celebrating)

### Current Implementation:
The images will cover the **entire screen** (full width and height) with text and buttons overlaid on top.

### Fallback:
If images are not found, the app will show a colored background as fallback.
